## Overview

This is a Cloudflare Worker expense-tracking app (with built-in authentication) using Hono, Drizzle ORM, Better Auth, Tailwind CSS, and DaisyUI.

## Directories

- `Notes/AGENTS.md` — all about the project documentation system
- `tests/` — unit tests
- `e2e-tests/` — end-to-end tests
- `src/` — source code
- `wrangler.toml` — Cloudflare Worker configuration
- `package.json` — project dependencies and scripts
